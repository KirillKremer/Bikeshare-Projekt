# Bikeshare Datenanalyse (Python)

## Beschreibung

In diesem Projekt habe ich ein Python-Programm entwickelt, das Bikeshare-Daten aus verschiedenen Städten analysiert. Ziel ist es, Nutzungsmuster zu erkennen und statistische Auswertungen durchzuführen.

## Technologien

* Python
* Pandas

## Datengrundlage

Das Projekt basiert auf Bikeshare-Daten aus mehreren Städten. Die Datensätze enthalten Informationen zu Fahrten, Zeitpunkten und Nutzungsverhalten.

## Ziel

Analyse von Nutzungsdaten und Identifikation von Mustern im Nutzerverhalten.

## Funktionsweise

Das Programm wird über die Konsole ausgeführt.
Der Nutzer kann folgende Filter auswählen:

* Stadt
* Monat
* Wochentag

Basierend auf diesen Eingaben werden die Daten gefiltert und analysiert.

## Analyse

Das Programm berechnet unter anderem:

* häufigsten Monat
* häufigsten Wochentag
* häufigste Startzeit
* weitere statistische Kennzahlen zur Nutzung

## Ergebnis

Durch die Analyse lassen sich typische Nutzungsmuster sowie Trends im Bikeshare-System erkennen.
